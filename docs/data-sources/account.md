# stalwart_account (Data Source)

Reads an existing account from Stalwart Mail Server. Credentials are never exposed — use the stalwart_account resource to manage passwords.

-> Credentials are never exposed by this data source. Use the `stalwart_account`
   resource to manage passwords.

## Example Usage

```terraform
data "stalwart_account" "alice" {
  name = "alice"
}

output "alice_primary_email" {
  value = data.stalwart_account.alice.primary_email
}

output "alice_member_of" {
  value = data.stalwart_account.alice.member_of
}
```

## Schema

### Required

- `name` (String) Account login name to look up.

### Read-Only

- `aliases` (List of String) Additional email addresses for this account.
- `description` (String) Display name or description for the account.
- `member_of` (List of String) Names of groups this account belongs to.
- `primary_email` (String) Primary email address (index 0 in Stalwart's emails array).
- `quota` (Number) Mailbox quota in bytes. 0 means unlimited.
- `roles` (List of String) Roles assigned to this account.
