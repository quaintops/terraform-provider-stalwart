# stalwart_dns_records (Data Source)

Reads the DNS records Stalwart requires for a given domain (MX, SPF, DKIM, DMARC, etc.).

-> Reference `stalwart_dkim.<name>.domain` as the `domain` argument rather
   than the domain name string directly. This creates an implicit dependency
   that ensures the DKIM key is generated before the DNS records are read,
   so the DKIM TXT record is included in the output.

## Example Usage

```terraform
resource "stalwart_domain" "example" {
  name = "example.com"
}

resource "stalwart_dkim" "example" {
  domain = stalwart_domain.example.name
}

data "stalwart_dns_records" "example" {
  domain = stalwart_dkim.example.domain
}

output "dns_records" {
  value = data.stalwart_dns_records.example.records
}
```
