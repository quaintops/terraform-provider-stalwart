# stalwart_group (Resource)

Manages a mail group (mailing list) in Stalwart Mail Server. Users join the group via their own member_of field, not the other way around. Mail sent to primary_email or any alias is delivered to all members.

-> **Membership is declared on the user side** via `stalwart_user.member_of`,
   not on the group. Groups must therefore be created before the users that join them.

-> When group email addresses come from a variable rather than a direct
   `stalwart_domain` resource reference, add `depends_on = [stalwart_domain.this]`
   so Terraform knows to create the domain first and destroy the group before the domain.

## Example Usage

```terraform
resource "stalwart_domain" "example" {
  name = "example.com"
}

resource "stalwart_group" "info" {
  name        = "info-example"
  description = "Info and hello group"

  primary_email = "info@${stalwart_domain.example.name}"
  aliases = [
    "hello@${stalwart_domain.example.name}",
    "contact@${stalwart_domain.example.name}",
  ]

  enabled_permissions = ["email-send", "email-receive"]

  external_members = ["partner@otherdomain.com"]

  depends_on = [stalwart_domain.example]
}
```

## Schema

### Required

- `name` (String) Internal name for this group (e.g. info-domain1). Must be unique across all principals.
- `primary_email` (String) Primary address for this group (e.g. info@domain1.com). Stored as index 0 in Stalwart.

### Optional

- `aliases` (List of String) Additional addresses that also deliver to this group (e.g. ["hello@domain1.com"]).
- `description` (String) Human-readable description.
- `disabled_permissions` (List of String) Permissions explicitly denied for this group.
- `enabled_permissions` (List of String) Permissions explicitly granted to this group (e.g. ["email-send", "email-receive"]).
- `external_members` (List of String) External email addresses that also receive mail sent to this group.
- `type` (String) Principal type: 'list' (mailing list, default) or 'group' (IMAP shared folder group).

### Read-Only

- `id` (String) Resource identifier. Always equal to name.

## Import

Groups can be imported using the group name:

```shell
terraform import stalwart_group.example info-example
```
